# Nail Segmentation Project

Welcome to our Nail Segmentation Project repository! This repository is your one-stop destination for all the resources and files necessary to dive into our innovative project. We've designed this project to leverage the power of AI for enhancing nail art techniques digitally. Our solution offers a range of features aimed at revolutionizing how nail aesthetics are approached, providing both professionals and enthusiasts with a toolkit to explore creativity.

## Key Features

### Nail Direction Detection
Utilizing the sophisticated hand landmark model from MediaPipe, we accurately identify fingertip and finger middle points. This crucial step allows us to determine the direction of the nail with precision. Understanding nail orientation is vital for applying intricate patterns or effects accurately, ensuring a flawless finish every time.

### Dynamic Nail Color Transformation
Ever thought of switching nail colors with just a click? Our feature empowers users to alter nail polish color virtually to any shade they desire. Through advanced image processing techniques, we ensure the color change looks natural and appealing, enriching the user experience by providing a real-time preview of varied aesthetic choices without the need for actual nail polish.

### Continuous Color Variation in Videos
Taking nail art visualization to the next level, this feature introduces the ability to dynamically change nail colors in video format. Imagine showcasing a range of nail colors or patterns over a video sequence - our technology makes this possible, providing a captivating and interactive way to display nail art possibilities.

## Setting Up the Project Locally

Before installing the dependencies, you'll need to clone the project repository to your local machine.

### Clone the Repository

To clone the repository, open your terminal and run the following command:

```sh
git clone https://github.com/ioptime-official/nails-segmentation.git
```
This command downloads the project files from GitHub to your computer, allowing you to work on the project locally.

### Install Dependencies

First, ensure you have Python installed on your system. Then, open your terminal and run the following command to install all required libraries:

```sh
pip install -r requirements.txt
```

This command reads the requirements.txt file and installs the Python packages necessary for running the project, ensuring a smooth setup process.
